# master-thesis-support-kit

修士論文の執筆および研究活動を総合的に支援するためのリソースを集約したリポジトリです。
修士課程の学生が、学術的誠実性を保ちつつ、効率的かつ高品質な論文を完成させるためのツールやガイドラインを提供します。

## リポジトリの目的

- 論文執筆プロセスの効率化
- 論理性・学術的妥当性のセルフチェック支援
- 執筆環境（LaTeX等）の構築サポート
- 研究倫理の遵守と適切なツール利用の指針提示

## 免責事項

本リポジトリで提供されるツールやプロンプトは執筆を「補助」するものであり、 **内容の正確性や学位授与を保証するものではありません** 。最終的な論文の責任はすべて執筆者本人に帰属します。

## 構成コンテンツ

本リポジトリは以下のコンテンツで構成されています。

### 1. ai-prompts
修士論文の各フェーズで活用できる AI プロンプト集です。
- 執筆文章の論理検証
- 説明の補足案・深化案の提示
- 数式画像や箇条書きの LaTeX 形式変換
- **利用前には必ず [ai-prompts/README.md](./ai-prompts/README.md) を一読してください。**

### 2. latex-guide
モダンな LaTeX 執筆環境を構築するためのガイドとサンプル集です。
- **[latex-guide/README.md](./latex-guide/README.md)**: 導入ガイドと信頼できる情報源（TeX Wiki）の紹介
- **エンジン選択**: LuaLaTeX をベースとした現代的な日本語環境の推奨
- **環境構築**: Windows/macOS でのローカル構築および最新クラウド環境（Prism, CloudLaTeX等）の比較
- **実践サンプル**: すぐに使える数学論文用最小テンプレート
- **発表スライド**: 修士論文の内容から Beamer で 20 分発表用スライドを作成するためのガイドとサンプル（[beamer-presentation-from-thesis.md](./latex-guide/beamer-presentation-from-thesis.md), [beamer-thesis-presentation.tex](./latex-guide/samples/beamer-thesis-presentation.tex)）


## 今後の拡張予定

本リポジトリは、必要に応じて以下のようなコンテンツを追加・更新していくかもしれません。

- 修士論文用 LaTeX テンプレート
- データ解析・可視化用スクリプト集（Python 等）
- 図表作成のガイドライン
- 引用文献管理（BibTeX）のベストプラクティス
