## 1. LaTeX とは
LaTeX（ラテフ / レイテック）は、論理構造に基づいた文書作成を行うための組版システムです。
- **数式の美しさ**: 数学・理系分野において世界標準の品質で数式を記述できます。
- **自動化**: 目次、図表番号、引用文献のリファレンスなどを自動的に管理します。
- **プレーンテキスト管理**: 執筆内容はテキストファイルとして保存されるため、Git 等による履歴管理と非常に相性が良いです。

## 3. 基本的な数式と図のサンプル
数学系論文で頻用される基本的な構造です。
実際にコンパイル可能なテンプレートは **[samples/README.md](./samples/README.md)** を参照してください。

### 数式の記述
複数行の数式を揃える場合は `align` 環境（amsmathパッケージ）を使用します。

```latex
\begin{align}
    f(x) &= \int_{a}^{b} g(t) \, dt \label{eq:integral} \\
    \sum_{n=1}^{\infty} \frac{1}{n^2} &= \frac{\pi^2}{6}
\end{align}
式\eqref{eq:integral}は...
```

### 図の挿入
図を配置する際は `figure` 環境を使用し、必ず `label` を付けて本文から参照します。

```latex
\begin{figure}[htbp]
    \centering
    \includegraphics[width=0.8\textwidth]{figures/sample-plot.pdf}
    \caption{数値実験の結果}
    \label{fig:result}
\end{figure}
図\ref{fig:result}に示す通り...
```

## 4. （参考）図の引用と文書装飾の作法

修士論文において、他者の文献から図を引用する場合や、文中の強調を行う際には、学術文書としての標準的なルールに従う必要があります。ここでは一般的な方法を述べますが、分野によっては異なる作法がある可能性があることは留意して下さい。

### 1. 図の引用・転載の表記
図を引用する際は、キャプション（`\caption`）の中に出典を明記します。図の状態（そのままか、加工したか）によって文言を使い分けます。

| ケース | 表記例 | LaTeXの実装例 |
| :--- | :--- | :--- |
| **そのまま転載** | [1] より引用 | `\caption{〇〇の構成（\protect\cite{key} より引用）}` |
| **一部改変・翻訳** | [1] を基に作成 | `\caption{〇〇の推移（\protect\cite{key} を基に作成）}` |

> **Tips:** `\caption` 内で `\cite` を使用してエラーが出る場合は、上記のように `\protect` を直前に置いてください。

> **Note:** 論文誌や学会に投稿する場合は、著作権者（出版社等）の転載許諾が必要な場合があります。修論においても、出典の明記は必須です。

### 2. 文中の強調（太字・下線の扱い）
学術論文では、過度な装飾は避け、論理構成で重要性を示します。

- **太字 (`\textbf{...}`)**:
  - 章・節の見出しや、定義・定理のラベル（**定義 1.1** など）に使用します。
  - 文中の強調として多用するのは避け、どうしても必要な箇所のみに留めます。
- **アンダーライン（下線）**:
  - **原則として使用しません。** 可読性を下げ、Webリンクと混同される恐れがあるためです。
- **等幅フォント (`\texttt{...}`)**:
  - プログラムの変数名や関数名（例：`predict()`）を文中で示す際に使用します。
- **数式の太字 (`\mathbf{...}`)**:
  - データサイエンス分野では、ベクトルや行列を太字で表記するのが一般的です（例：$\mathbf{x}, \mathbf{A}$）。
