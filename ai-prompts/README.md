# AI プロンプト集

修士論文執筆の各段階で活用できる AI プロンプトのテンプレート集です。

## はじめに

**重要な前提：AI 利用の基本方針**

このディレクトリのプロンプトは、修士論文の執筆や準備を補助するためのツールです。  
AI はあくまで「論理チェック」「説明案の提示」「形式変換」といった補助的な役割を果たすものであり、研究の独自性や学術的誠実性を損なわない範囲で使用してください。

- **内容生成を丸投げしない**：AI に論文の本文や主張を書かせるのではなく、自分で書いた文章の改善や検証に使う。
- **引用・先行研究の確認は必須**：AI が提示する情報は不正確な場合があるため、学術文献で必ず裏取りをする。
- **指導教員や所属機関の規定を優先**：AI 利用のガイドラインがある場合は、それに従う。

## ファイル構成

このディレクトリには、以下のテンプレートファイルが含まれています。

| ファイル名 | 説明 |
|-----------|------|
| `01_master_thesis_ai_usage_principles.md` | AI を利用する際の基本原則と注意点をまとめたガイドライン。必ず最初に読むこと。 |
| `02_master_thesis_analysis_templates.md` | 既存の文章を論理的に分析・検証するためのプロンプト集。 |
| `03_master_thesis_latex_conversion.md` | 数式画像や箇条書きを LaTeX 形式に変換するためのプロンプト。 |
| `04_master_thesis_explanation_expansion.md` | 説明を補足・深化させるためのプロンプト。読者の理解を助ける表現を検討する際に使用。 |
| `05_master_thesis_public_hearing_support.md` | 公聴会（修士論文発表）の質問シミュレーションと回答評価に使うテンプレート。 |

## 使い方

1. **`01_master_thesis_ai_usage_principles.md` を必ず最初に読む**  
   AI 利用の基本方針と、やってはいけないことを理解してください。

2. **目的に合ったテンプレートを選ぶ**  
   各ファイルには、具体的な利用シーンごとに複数のプロンプトテンプレートが含まれています。

3. **プレースホルダ（`[...]`）を埋める**  
   テンプレート内の `[研究テーマ]` `[対象読者]` などの部分を、自分の状況に合わせて書き換えてください。  
   ファイルを添付する場合は、プレースホルダに何も書かれていない情報を AI が添付ファイルから読み取るように指示できます。

4. **AI に入力して結果を検証する**  
   出力された内容は必ず自分で確認し、誤りや不適切な箇所があれば修正してください。

## 注意事項

- **AI の出力は必ずしも正確ではない**：特に、専門的な内容や最新の研究に関する情報は誤りを含む可能性があります。
- **学術的誠実性を最優先**：AI が生成した文章をそのまま論文に使用することは、剽窃とみなされる場合があります。
- **プライバシーとセキュリティ**：未公開の研究データや個人情報を AI サービスに入力する際は、利用規約を確認してください。
